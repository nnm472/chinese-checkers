# 中国のチェッカー

スマホ縦向きを中心にした、おはじき物理ボードゲームです。

## 遊び方

- 2人対戦では、赤と緑が1台の端末で向き合って交互にプレイします。
- 1人プレイでは、赤を操作して緑CPUと対戦します。
- 自分の色のコマを選んで引っ張り、指を離すと反対方向へ飛びます。
- 上下左右どこからでも外側の落下ラインを越えたコマは除去されます。
- 中央付近の水平な反射板だけが跳ね返します。
- 相手のコマをすべて落としたプレイヤーの勝利です。

## 現在の実装

- 赤先攻
- 各チーム: 大1個 / 中8個 / 小7個
- 特殊能力: Power / Skip / Grow / Steal
- スキルポイント: 手番終了で+1、自分の小/中/大/特大コマ落下で+2/+4/+6/+6
- 1人プレイ時、CPUはPowerのみ使用し、ポイントが多い時はほぼ必ず使用します。
- CPUの発射前に、方向と強さのプレビューが短く表示されます。
- CPUは通常時も強めに打ち、端に近い相手コマを狙いやすくしています。
- 小コマは発射・衝突時の速度を少し控えめに調整しています。
- 勝利時は盤面を少し見せてから、盤面を隠さない結果表示を出します。
- チュートリアル、設定、リスタート、勝敗画面あり
